#  SillyTavern Extension - Lorebook Profiles

Save and activate named lorebook combinations with one click.

Instead of manually toggling lorebooks every time you switch characters
or scenarios, you create a profile once and apply it instantly.

## Installation

1. In SillyTavern, go to **Extensions → Install Extension**
2. Paste: `https://github.com/RPFiend/ST-Lorebooks-Extended`
3. Restart/Refresh SillyTavern

## Usage

**Create a profile** — Enter a name, check the lorebooks to include, hit Save.

**Activate a profile** — Click Activate next to any saved profile.
All lorebooks in the profile turn on; everything else turns off.

**Delete a profile** — Click the ✕ button next to it.

## Notes

- Profiles persist across restarts via SillyTavern's built-in settings
